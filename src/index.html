Esta PWA (Aplicación Web Progresiva) tiene como objetivo implementar un modelo de Machine Learning (ML) para la obtención de un pre-diagnostico que ayuda a detectar la Enfermedad de Parkinson, mediante un test de la voz.

Es necesario grabar un pequeño test de audio para después mediante la aplicación recibir un resultado del análisis de este, respecto a la voz.
Para ello, realizar las siguientes recomendaciones e instrucciones para grabar el audio:

Recomendaciones:

  • Buscar un lugar libre de ruido y distracciones.
  • Tomar un poco de agua.
  • Evitar grabar enfermo de la garganta.
  • Seguir las instruciones de forma ordenada.

Instrucciones:

  1. Buscar un lugar libre de ruido y distracciones.
  2. Tomar un poco de agua.
  3. Evitar grabar enfermo de la garganta.
  4. Seguir las instruciones de forma ordenada.

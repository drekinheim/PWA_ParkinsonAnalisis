Esta PWA (Aplicación Web Progresiva) tiene como objetivo implementar un modelo de Machine Learning (ML) para la obtención de un pre-diagnostico que ayuda a detectar la Enfermedad de Parkinson, mediante un test de la voz.

Es necesario grabar un pequeño test de audio para después mediante la aplicación recibir un resultado del análisis de este, respecto a la voz.
Para ello, realizar las siguientes recomendaciones e instrucciones para grabar el audio:

Recomendaciones:

  • Buscar un lugar libre de ruido y distracciones.
  • Tomar un poco de agua.
  • Evitar grabar enfermo de la garganta.
  • Seguir las instruciones de forma ordenada.

Instrucciones:

  1. Decir la letra A durante 3 segundos.
  2. Descansar 3 segundos.
  3. Decir la letra A durante 3 segundos.
  4. Descansar 3 segundos.
  5. Decir la letra A durante 3 segundos.
  6. Descansar 3 segundos.
  7. Decir la letra O durante 3 segundos.
  8. Descansar 3 segundos.
  9. Decir la letra O durante 3 segundos.
  10. Descansar 3 segundos.
  11. Decir la letra O durante 3 segundos.

En resumen, pronunciaras la letra A, sin interucciones por 3 segundos, 3 veces, y entre cada repetición se dejará un lapso de 3 segundos. Después, pronunciaras la letra O, sin interrupciones por 3 segundos, 3 veces, y entre cada repetición se dejará un lapso de 3 segundos hasta la penúltima repetición. La grabación tendrá una duración total apróximada de 33 segundos.
